Додаток для реєстрації на події та візуалізації статистики.

## Технології:
- React + Vite
- Redux Toolkit (State Management)
- Recharts (Data Visualization)
- Vercel (Hosting)

## Як запустити:
1. `npm install`
2. `npm run dev` - фронтенд на http://localhost:5173

## Сервер (ПР №3):
1. `npm run server:dev` - сервер на http://localhost:3000

### Маршрути:
- `GET /api/events` - список усіх подій
